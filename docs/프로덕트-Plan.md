# 썬드림 룰렛 상용화(Product) 기획서

## 1. 개요
현재 단일 이벤트 목적으로 개발된 썬드림 룰렛 엔진을, 범용적이고 수익 창출이 가능한 **'SaaS형 실시간 이벤트 플랫폼'**으로 고도화하기 위한 중장기 로드맵입니다.

## 2. 핵심 목표 (Product Vision)
1. **B2B / B2C 확장**: 스트리머, 유튜버, 기업 행사 담당자가 독립적인 방을 파서 이벤트를 진행할 수 있는 다중 테넌트(Multi-Tenant) 구조를 확립합니다.
2. **과금 모델 도입**: 이벤트 동시접속자 수, 사은품 규모 등에 따른 티어별 구독 또는 건별 결제 시스템을 도입합니다.
3. **사용성 극대화**: 개발자나 관리자의 개입 없이 누구나 쉽게 이벤트를 생성하고 커스터마이징할 수 있는 직관적인 어드민 빌더를 제공합니다.

## 3. 고도화 개발 과제 (Roadmap)

### 단계 1: 다중 테넌트 및 계정 시스템 (Phase 1)
- [ ] **회원가입 및 계정 관리**: JWT 기반 사용자 인증 시스템 (Google, Naver 등 소셜 로그인 지원).
- [ ] **방(Room) 단위 이벤트 생성**: URL 해시(예: `/room/abc1234`) 기반의 독립된 게임 채널링 로직 구현.
- [ ] **관리자 권한 분리**: 마스터 어드민(전체 통계) / 룸 어드민(해당 방 제어) 권한 체계 분리.

### 단계 2: 어드민 대시보드 및 통계 강화 (Phase 2)
- [ ] **GUI 기반 커스터마이징**: 룰렛 원판 색상, 팡파레 효과음, 배경 이미지 등을 웹에서 직접 업로드하고 설정할 수 있는 빌더 기능.
- [ ] **상세 통계 대시보드**: 일일 트래픽, 최고 동접자 수, 당첨자 이력 로그 CSV 다운로드 기능.
- [ ] **모바일 관리 앱 (또는 PWA)**: PC가 없는 환경에서도 긴급 제어가 가능한 태블릿/모바일 전용 최적화 어드민 콘솔 구축.

### 단계 3: 플랫폼 확장 및 연동 (Phase 3)
- [ ] **외부 플랫폼 연동 API (Webhook)**: 유튜브 슈퍼챗, 아프리카TV 별풍선, 트위치 후원 시 자동으로 룰렛 명단에 티켓이 추가되는 연동 API 개방.
- [ ] **보안 및 트래픽 분산 (Scale-Out)**: Redis Pub/Sub을 활용한 다중 Socket.IO 노드 구성으로 1만 명 이상의 메가 이벤트 수용 기반 마련.
- [ ] **어뷰징 방지**: 매크로 및 비정상적 트래픽 차단을 위한 IP 밴, 캡챠 도입 로직 구현.

## 4. 모니터링 및 유지보수 계획
- **서버 모니터링**: Datadog 또는 Prometheus/Grafana를 도입하여 실시간 소켓 연결 수 및 DB 병목 모니터링.
- **무중단 배포(CI/CD)**: 상용화 단계에서는 Render의 기본 플랜을 벗어나 AWS/GCP 기반의 무중단 배포 파이프라인 구축.
